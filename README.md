## Banco

Projeto para praticar Orientação a Objetos com Java

Nesse projeto eu estou colocando em prática a orientação a objetos com Java, onde o Objetivo é criar o Back-end de uma aplicação de Banco que tenha as funcionalidades de Sacar, Depositar, Verificar Saldo e Transferir para outra Conta

É um projeto simples de prática
